# 📦 StockAtelier - Gestion de Stock Café

**StockAtelier** est une application web moderne et minimaliste conçue pour aider les propriétaires de petits cafés à gérer leur inventaire en temps réel. Développée avec **Next.js 15**, **Tailwind CSS**, et **Supabase**.

---

## 🌟 Fonctionnalités Principal
* **Dashboard Intuitif** : Vue d'ensemble sur la valeur totale du stock et le nombre d'articles.
* **Gestion CRUD Complet** : Ajouter, modifier, et supprimer des produits facilement.
* **Real-time Sync** : Synchronisation instantanée avec la base de données Supabase.
* **Interface "Clean"** : Design épuré avec un mode sombre/clair élégant.

## 🛠️ Stack Technique
* **Framework** : [Next.js 15](https://nextjs.org/) (App Router)
* **Base de Données** : [Supabase](https://supabase.com/) (PostgreSQL)
* **Styling** : [Tailwind CSS](https://tailwindcss.com/)
* **Déploiement** : [Vercel](https://vercel.com/)
